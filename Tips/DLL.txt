Visual C++ 支援三種 DLL [1] ：

Non-MFC DLL - 直接用 C 語言寫的 DLL ，導出函數為標準的 C 接口，非 MFC 或 MFC 程式皆可使用
Regular DLL - 用 MFC 編寫，導出函數可以是 C / C++類別 / C++成員函數，非 MFC 或 MFC 程式皆可使用
Extension DLL - 用 MFC 編寫，導出 MFC 類別，只能被 MFC 程式使用
